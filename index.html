بنية الثقة المعدومة (Zero-Trust)

تم بناء "حِصْن" كملف استاتيكي يعمل محلياً بمتصفحك دون إرسال بايت واحد عبر الشبكة (Offline First).

  • تشفير مضاعف: AES-GCM 256 بمفاتيح مشتقة عبر PBKDF2 بـ مليون دورة.
  • تدمير المراقبة: انتقالك لتطبيق آخر أو ضغطك لزر الطوارئ ينفذ تدميراً فورياً للذاكرة العشوائية.
  • تصفير عميق (Zero-fill): كتابة أصفار 0x00 فوق بيانات الـ RAM لتجنب برمجيات فحص الذاكرة.
  • حماية الصور: ترسم الصور كبيكسلات على Canvas، وتباد الروابط الأصلية لحظياً، مع إيقاف قوائم الضغط المطول بالهواتف.
يعمل محلياً بمعيار Web Crypto API النقي.
